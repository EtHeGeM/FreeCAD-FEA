# Shaft 6061-T6 FreeCAD FEM

This project contains `shaft_6061_fem.py`, a terminal-driven FreeCAD FEM automation script for a linear static cantilever shaft analysis.

## Requirements

- Linux/Ubuntu
- FreeCAD with FEM Python modules
- Gmsh for meshing: `sudo apt install gmsh`
- CalculiX solver: `sudo apt install calculix-ccx`

## Run

From this directory:

```bash
FreeCADCmd shaft_6061_fem.py
```

or:

```bash
freecadcmd shaft_6061_fem.py
```

The script saves:

- `shaft_6061_t6_fem.FCStd`
- solver files under `shaft_6061_fem_output/`
- CalculiX `.inp`, `.frd`, `.dat`, or result files when generated by the installed FreeCAD/CalculiX stack
- `.vtu` export when the installed FreeCAD VTK result exporter is available

## Open Results

Open `shaft_6061_t6_fem.FCStd` in the FreeCAD GUI to inspect the geometry, constraints, mesh, solver, and result objects.

If a `.vtu` file is created in `shaft_6061_fem_output/`, open it directly in ParaView.

## Debugging

If `FreeCADCmd` is missing, check the installed binary name:

```bash
which FreeCADCmd
which freecadcmd
```

If neither exists, install FreeCAD or add its binary directory to `PATH`.

If Gmsh is missing:

```bash
sudo apt install gmsh
which gmsh
```

If CalculiX is missing:

```bash
sudo apt install calculix-ccx
which ccx
```

The script prints every detected face name and center of mass. If a face detection error occurs, use that table to verify that the geometry contains circular faces at `x = 0`, `x = 400`, and `x = 600` mm.
